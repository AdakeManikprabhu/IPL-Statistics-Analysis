# IPL Team Performance Analysis

## Project Overview
This project analyzes IPL data from 2008 to 2024 using Python.

## Features
- Team performance analysis
- Auction data analysis
- Ball-by-ball statistics
- Data visualization
- Performance comparison between teams

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
- IPL Ball By Ball Data (2008-2024)
- Team Performance Dataset
- Auction Dataset
